# Создание файла .md (Markdown): назначение и основы

## 1. Что такое файл .md
Файл с расширением `.md` — это текстовый файл, созданный с использованием языка разметки **Markdown**.

Markdown применяется для:
- подготовки учебной и технической документации;
- оформления отчётов, заданий и методических материалов;
- создания веб-документации (MkDocs, GitHub Pages).

---

## 2. Создание файла .md
Файл `.md` создаётся как обычный текстовый файл:
- через проводник операционной системы;
- в редакторе кода (VS Code, PyCharm и др.);
- через терминал.

Пример имени файла: `introduction.md`

---

## 3. Основные элементы Markdown

### Заголовки
```markdown
# Заголовок 1 уровня
## Заголовок 2 уровня
### Заголовок 3 уровня
```

### Выделение
```markdown
**Жирный текст**
*Курсивный текст*
```

### Списки
```markdown
- первый пункт
- второй пункт

1. первый пункт
2. второй пункт
```

### Блоки кода
````markdown
```python
print("Hello, Markdown")
```
````
