# Описание файла mkdocs.yml

Файл `mkdocs.yml` является основным конфигурационным файлом системы документации MkDocs.

Он определяет:
- название и назначение сайта;
- используемую тему оформления;
- плагины;
- структуру навигации;
- подключаемые расширения Markdown;
- дополнительные стили и скрипты.

> Требуется хранить `mkdocs.yml` в корне репозитория.
