    Открытие папки проекта через терминал или IDE

    Вариант 1. Через IDE (рекомендуется)

    Рекомендуемая среда разработки: VS Code.

    1. Откройте VS Code
    2. Выберите File → Open Folder... и укажите папку проекта
    3. Откройте встроенный терминал: Terminal → New Terminal
    4. Убедитесь, что текущая директория терминала — корень проекта

    Вариант 2. Через терминал

    1. Откройте терминал
    2. Перейдите в папку проекта командой cd:
    cd путь/к/папке/проекта

    Все дальнейшие команды следует выполнять из корня репозитория/проекта, если не указано иное.
