
# Создание скриншотов в VS Code с сохранением в папку проекта

## Назначение

Скриншоты можно создавать сразу для работы с приложением

---

## Рекомендуемая структура для скриншотов

В проекте MkDocs рекомендуется использовать папку:

```text
docs/
└─ assets/images/
```

## Подготовка папки в VS Code

1. Откройте проект в **VS Code**
2. В панели **Explorer**:

    * нажмите правой кнопкой на папке `img`
    * выберите **New Folder**
    * создайте папку `screenshots`
3. Внутри `screenshots` создайте нужные подпапки (например, `vscode`)

> Папки должны быть созданы **до** выполнения скриншотов.

---

## Создание скриншота (Windows)

### Способ 1. Win + Shift + S (рекомендуется)

1. Нажмите **Win + Shift + S**
2. Выберите тип снимка:

    * прямоугольная область (рекомендуется);
    * окно;
    * весь экран

3. Выделите нужную область (например, окно VS Code)

4. Скриншот будет помещён в буфер обмена

### Сохранение сразу в нужную папку

1. Перейдите в VS Code
2. В панели **Explorer** откройте папку:

   ```text
   docs/img/screenshots/vscode
   ```
3. Нажмите **Ctrl + V**

4. VS Code автоматически предложит имя файла — переименуйте его осмысленно, например:

   ```text
   vscode-project-structure.png
   ```

> Таким образом, скриншот **сразу сохраняется в нужную папку проекта**.

---

## Создание скриншота (macOS)

1. Нажмите **Cmd + Shift + 4**
2. Выделите область экрана
3. По умолчанию файл сохраняется на рабочий стол

### Перемещение в папку проекта

Рекомендуется:

* сразу перетащить файл в папку
  `docs/img/screenshots/vscode`
* либо изменить путь сохранения через:

  ```bash
  defaults write com.apple.screencapture location путь_к_проекту/docs/img/screenshots
  ```
