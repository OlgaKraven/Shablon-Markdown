# Создание репозитория GitHub

## Порядок действий

1. Перейдите во вкладку **Repositories**
2. Нажмите **New**

![Пример страницы](../assets/images/2.png)
/// caption
Рисунок 1 — Пример страницы Repositories
///

3. Укажите:
      - имя репозитория;
      - краткое описание проекта;
4. Создайте репозиторий

![Пример создания репозитория](../assets/images/3.png)
/// caption
Рисунок 2 — Создание репозитория
///


## Локальная папка проекта

В удобном месте на компьютере создайте папку **с тем же названием**, что и репозиторий GitHub.

> Это упростит синхронизацию и исключит путаницу при работе с несколькими проектами.


![Пример папки](../assets/images/4.png)
/// caption
Рисунок 3 — Пример папки
///

## Если требуется снять копию проекта

В необходимом месте для сохранения вибераем папку, которая будет корневой для данного проекта
и пишем git clone *Название репозитория*

Название репозитория берется из проекта см. рисунок 4.
