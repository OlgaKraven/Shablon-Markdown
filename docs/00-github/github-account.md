# Создание аккаунта GitHub и авторизация

## Назначение
GitHub используется для хранения исходного кода, документации и контроля версий учебного проекта.

## Шаги

1. Перейдите на сайт [Официальный сайт github](https://github.com/)
2. Создайте аккаунт или авторизуйтесь через электронную почту от google.
3. После входа убедитесь, что доступна вкладка **Repositories**
