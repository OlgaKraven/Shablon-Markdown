# Навигация (nav) в MkDocs

Раздел `nav` задаёт логическую структуру документации и порядок отображения пунктов меню.

## Рекомендации

- группируйте страницы по разделам (GitHub, Python, MkDocs, Плагины, Конфигурация, Markdown);
- используйте предсказуемые пути и имена файлов;
- не делайте меню чрезмерно глубоким — лучше больше страниц и логичнее структура.

Пример:

```yaml
nav:
  - Главная: index.md
  - GitHub:
      - Аккаунт: 00-github/github-account.md
      - Репозиторий: 00-github/github-repository.md
```
