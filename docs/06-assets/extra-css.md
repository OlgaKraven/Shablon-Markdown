# Создание файла пользовательских стилей extra.css

Для настройки внешнего вида сайта документации MkDocs Material используется отдельный файл пользовательских стилей `extra.css`, который позволяет переопределять стандартные стили темы без изменения её исходного кода.

---

## 1. Назначение файла extra.css

Файл применяется для:
- настройки цветовой схемы проекта;
- адаптации интерфейса под учебные требования;
- улучшения читаемости;
- стилизации Mermaid и блоков цитат.

---

## 2. Размещение файла в проекте

```text
docs/
└─ stylesheets/
   └─ extra.css
```

---

## 3. Подключение файла в mkdocs.yml

```yaml
extra_css:
  - stylesheets/extra.css
```

---

## 4. Содержимое файла extra.css

См. файл: `docs/stylesheets/extra.css`
