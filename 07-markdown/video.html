    Добавление видео в markdown

    1. Встраивание видео через HTML <video> (РЕКОМЕНДУЕТСЯ для локальных файлов)

    Markdown сам по себе не поддерживает видео, но MkDocs разрешает HTML внутри .md, поэтому используется тег <video>.

    Пример (локальный файл)

    <video controls width="100%">
  <source src="/assets/videos/1.mp4" type="video/mp4">
  Ваш браузер не поддерживает воспроизведение видео.
</video>

    Рекомендуемая структура

    docs/
└─ assets/
   └─ videos/
      └─ 1.mp4

    Пояснение

    • controls — кнопки воспроизведения
    • width="100%" — адаптация под ширину страницы
    • видео хранится в репозитории, подходит для офлайн-проверки

    2. Видео с подписью (caption) — учебный формат

    Если вы используете pymdownx.blocks.caption, подпись добавляется так же, как для таблиц и изображений:

    <video controls width="100%">
  <source src="assets/videos/mkdocs-serve.mp4" type="video/mp4">
</video>

/// caption
Видео 1 — Запуск локального сервера MkDocs
///

    ✔ соответствует учебным требованиям ✔ единый стиль с рисунками и таблицами

    3. Встраивание видео с YouTube (через <iframe>)

    Пример

    <iframe
  width="100%"
  height="400"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube video player"
  frameborder="0"
  allowfullscreen>
</iframe>

    Когда использовать

    • видео слишком большое для репозитория;
    • материал размещён публично;
    • допустима зависимость от интернета.

    4. Ссылка на видео (простой и безопасный вариант)

    [Смотреть видео: Демонстрация работы MkDocs](assets/videos/demo.mp4)

    или

    [Смотреть видео на YouTube](https://youtu.be/VIDEO_ID)

    Подходит, если:

    • видео — дополнительный материал;
    • встраивание не требуется.

    5. Что НЕ работает и почему

    ❌

    ![video](video.mp4)

    → Markdown считает это изображением, видео не воспроизводится.

    ❌

    <video src="video.mp4">

    → без <source> часто не работает корректно.

    6. Рекомендации для учебных проектов

    • Используйте формат .mp4 (H.264) — максимальная совместимость

    • Размер видео:

    • желательно до 50–100 МБ

    • Для отчётов:

    • добавляйте подпись (caption);

    • упоминайте видео в тексте (“см. видео 1”)
    • Храните видео в docs/assets/videos/

    7. Минимальный рабочий пример (копировать и использовать)

    <video controls width="100%">
  <source src="assets/videos/example.mp4" type="video/mp4">
</video>

/// caption
Видео 1 — Демонстрация структуры проекта MkDocs
