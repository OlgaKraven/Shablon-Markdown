# Виртуальное окружение Python (ОЧЕНЬ ВАЖНО)

## Зачем нужно виртуальное окружение

Виртуальное окружение изолирует зависимости проекта (версии библиотек), чтобы:
- разные проекты не конфликтовали по версиям пакетов;
- настройка окружения была воспроизводимой;
- проверка и сдача работ проходили стабильно.

---

## Создание окружения

В корне проекта:

```bash
python -m venv .venv
```

---

## Активация окружения

### Windows

```bash
.venv\Scripts\activate
```

### macOS / Linux

```bash
source .venv/bin/activate
```

---

## Установка пакетов внутри окружения

```bash
pip install mkdocs-material material-glightbox
```

---

## Деактивация окружения

```bash
deactivate
```
